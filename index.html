About

Здравствуйте, я верстальщик и начинающий веб-разработчик.

Могу сверстать макеты низкой, средней и выше среднего сложностей, оптимизировать сайт (pagespeed).

Вёрстка адаптивна (при наличии соответсвующего макета) и кроссбраузерна.

Для вёрстки использую HTML, CSS, SASS, JS, JQuery.

HTML

HTML - это не язык программирования, это язык гипертекстовой разметки, он используетя для сообщения браузеру, как отображать веб-страницы, которые вы посещаете. Он может быть сложным или простым, в зависимости от желания веб-дизайнера. HTML состоит из ряда элементов, которые используются для вложения и оборачивания различных частей контента, для его отображения и различного поведения.

CSS

Как и HTML, CSS на самом деле не является языком программирования, это язык таблицы стилей. Это означает, что он позволяет выборочно применять стили к элементам в документах HTML, задавая элементам например: цвета, размеры, позиционирование и определенное поведение.

Sass

Sass — это метаязык (язык для описания другого языка), который упрощает и ускоряет написание CSS-кода. Его часто называют препроцессором CSS — это означает, что Sass имеет свой синтаксис, из которого генерирует CSS-код, понятный любому браузеру.

JavaScript

JavaScript это язык, который позволяет вам применять сложные вещи на web странице — каждый раз, когда на web странице происходит что-то большее, чем просто её статичное отображение — отображение периодически обновляемого контента, или интерактивных карт, или анимация 2D/3D графики, или прокрутка видео в проигрывателе, и т.д. — можете быть уверены, что скорее всего, не обошлось без JavaScript.

JQuery

jQuery — набор функций JavaScript, фокусирующийся на взаимодействии JavaScript и HTML. Библиотека jQuery помогает легко получать доступ к любому элементу DOM, обращаться к атрибутам и содержимому элементов DOM, манипулировать ими. Также библиотека jQuery предоставляет удобный API для работы с AJAX.

Github

GitHub — это сервис для совместной разработки и хостинга проектов. C помощью GitHub над кодом проекта может работать неограниченное количество программистов из любых точек мира. В GitHub есть система контроля (управления) версий Git: сервис позволяет просматривать и контролировать любые изменения кода любым разработчиком и возвращаться к состоянию до изменений.

Git

Git — система управления версиями с распределенной архитектурой. В отличие от некогда популярных систем вроде CVS и Subversion (SVN), где полная история версий проекта доступна лишь в одном месте, в Git каждая рабочая копия кода сама по себе является репозиторием. Это позволяет всем разработчикам хранить историю изменений в полном объеме.

Figma

Figma — это графический онлайн-редактор для совместной работы. В нём можно создать прототип сайта, интерфейс приложения и обсудить правки с коллегами в реальном времени.

VS Code

Visual Studio Code (VS Code) — редактор исходного кода, разработанный Microsoft для Windows, Linux и macOS. Позиционируется как «лёгкий» редактор кода для кроссплатформенной разработки веб- и облачных приложений. Включает в себя отладчик, инструменты для работы с Git, подсветку синтаксиса, IntelliSense и средства для рефакторинга. Имеет широкие возможности для кастомизации: пользовательские темы, сочетания клавиш и файлы конфигурации.
